## Sumário Executivo

### Resumo do Negócio

A ONG "Amigos dos Animais" busca criar um aplicativo inovador que facilitará a adoção de animais de estimação, conectando animais resgatados com pessoas interessadas em adotá-los. O aplicativo funcionará como uma plataforma centralizada para encontrar, avaliar e adotar animais de forma responsável.

### Missão

Promover a adoção responsável de animais de estimação, reduzindo o abandono e contribuindo para o bem-estar animal.

### Visão

Tornar-se a principal plataforma de adoção de animais no Brasil, promovendo uma cultura de cuidado e responsabilidade para com os animais de estimação.

## Análise de Mercado

### Tendências de Mercado

- A crescente conscientização sobre o bem-estar animal está aumentando a demanda por adoção de animais de estimação.
- O uso de aplicativos para a adoção de animais está se tornando mais comum em todo o mundo.
- A pandemia de COVID-19 intensificou o interesse pelas adoções de animais de estimação.

### Público-Alvo

- Pessoas interessadas em adotar animais de estimação.
- ONGs e abrigos de animais que desejam listar animais disponíveis para adoção.

### Estratégia de Marketing

#### Posicionamento

- Oferecer uma plataforma intuitiva e segura para encontrar o animal de estimação perfeito.
- Promover a ideia de adoção responsável e a importância de cuidar dos animais.

#### Estratégias de Marketing

- Campanhas nas redes sociais, com foco em histórias de sucesso de adoções.
- Parcerias com ONGs e abrigos de animais para listar seus animais no aplicativo.
- Programas de fidelidade para adotantes frequentes.
- Promoção de eventos de adoção em parceria com lojas de animais.

## Plano Operacional

### Desenvolvimento do Aplicativo

- Contratar uma equipe de desenvolvedores de aplicativos ou terceirizar o desenvolvimento.
- Implementar recursos como perfis de animais, fotos, vídeos e informações sobre saúde.

### Cadastro de Animais

- Coletar informações detalhadas sobre os animais disponíveis para adoção.
- Fornecer serviços de verificação de saúde e vacinação.

### Funcionalidades Adicionais

- Sistema de mensagens para comunicação entre adotantes e abrigos.
- Recursos de compartilhamento nas redes sociais para aumentar a visibilidade dos animais disponíveis.

## Plano Financeiro

### Fontes de Receita

- Taxa de inscrição para ONGs e abrigos que desejam listar seus animais.
- Taxa de assinatura mensal para adotantes em busca de recursos adicionais.
- Parcerias com empresas locais para anúncios e promoções.

### Projeção Financeira

- Estimativa de custos iniciais de desenvolvimento do aplicativo.
- Projeção de receitas com base nas taxas de inscrição e assinaturas.
- Monitoramento constante dos custos operacionais e ajustes conforme necessário.

## Responsabilidade Social e Ética

- Priorizar o bem-estar dos animais em todos os aspectos do negócio.
- Promover a esterilização e castração como parte do processo de adoção.
- Estabelecer diretrizes rigorosas para garantir a integridade do processo de adoção.

## Conclusão

A criação de um aplicativo para facilitar a adoção de animais é uma iniciativa louvável que pode ter um impacto significativo na vida dos animais e das pessoas. Com um plano de negócios sólido, comprometimento com o bem-estar animal e uma estratégia de marketing eficaz, a ONG "Amigos dos Animais" está bem posicionada para alcançar seus objetivos e criar um impacto positivo na sociedade.


## Funcionalidades

- **Cadastro de Animais:** Abigos de animais podem cadastrar informações detalhadas sobre os animais disponíveis para adoção, incluindo fotos, descrições e requisitos especiais.

- **Pesquisa Avançada:** Os usuários podem buscar animais com base em critérios específicos, como raça, tamanho, idade e localização.

- **Solicitação de Adoção:** Os interessados podem enviar solicitações de adoção para os abrigos de animais.

- **Painel de Administração:** Abigos de animais têm acesso a um painel de administração onde podem gerenciar informações sobre animais, aprovar solicitações de adoção e administrar usuários.

- **Autenticação e Autorização:** O sistema oferece autenticação segura e autorização com diferentes níveis de acesso para administradores e usuários comuns.

## Instalação

Siga as etapas abaixo para instalar e executar o projeto:

1. **Clone o repositório:**

 ```bash
 git clone https://github.com/seu-usuario/AdoteAi.git
 ```
   
Navegue até o diretório do backend:
```bash
cd AdoteAi/Backend
```

Execute o servidor ASP.NET Core:
```bash
dotnet run
```

Navegue até o diretório do frontend:
```bash
cd ../Frontend
```

Instale as dependências do Angular:
```bash
npm install
```

Inicie o servidor de desenvolvimento do Angular:
```bash
ng serve
```
Agora você pode acessar o aplicativo em http://localhost:4200 no seu navegador.
